# Welcome to your Lovable project

This repository contains the wedding invitation app for `sravaniwedsraju.online`.

## Deployment

This project is configured to deploy automatically to GitHub Pages when `main` is pushed.

- A custom domain is set in `public/CNAME`.
- GitHub Actions workflow is defined in `.github/workflows/gh-pages.yml`.

To publish:

1. Push your changes to `main`.
2. In GitHub repository settings, enable Pages from the `gh-pages` branch.
3. Ensure the domain `sravaniwedsraju.online` is pointed at GitHub Pages.

If you prefer Vercel or Netlify instead, I can also help add that configuration.
